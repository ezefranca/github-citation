How this citation was generated

This workspace reports whether the result came from CITATION.bib, CITATION.cff, or an inferred GitHub metadata fallback.

  1. Use CITATION.bib directly if the repository ships maintainer-authored BibTeX.
  2. Convert CITATION.cff into BibTeX when a citation file exists but is not already in BibTeX format.
  3. If neither file exists, generate a reviewable @misc entry from GitHub repository metadata.

Citation basics

What is a citation?

A citation is a way of giving credit to individuals for their creative and intellectual works that you utilized to support your research. It can also be used to locate particular sources and combat plagiarism. Typically, a citation can include the author's name, date, location of the publishing company, journal title, or DOI (Digital Object Identifier).

What is a citation style?

A citation style dictates the information necessary for a citation and how the information is ordered, as well as punctuation and other formatting.

  • APA (American Psychological Association): Used in the social sciences (e.g., psychology, education, business). It emphasizes the publication date to show the chronology of research.
  • MLA (Modern Language Association): Used in the humanities (e.g., literature, language, cultural studies). It emphasizes page numbers so readers can easily locate exact quotes.
  • Chicago Manual of Style: Commonly used in history and the arts. It is highly flexible and offers two formats: Notes and Bibliography (using footnotes/endnotes) or Author-Date.
  • IEEE: Used in engineering, computer science, and information technology. It uses bracketed numbers in the text (e.g., [1]) tied to a numbered reference list.
  • Harvard: A widely used author-date system, very similar to APA, that is commonly used in the UK and Australia.
